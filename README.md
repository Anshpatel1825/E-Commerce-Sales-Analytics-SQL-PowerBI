# 🛒 E-Commerce Sales & Performance Analysis

## 📌 Project Overview 

This project focuses on analyzing an E-Commerce dataset to extract meaningful business insights.  
The project covers the complete data analysis lifecycle including data cleaning, transformation, modeling, and dashboard creation.

The main goal is to understand customer behavior, sales performance, delivery efficiency, and seller contribution using real-world data.

## 🎯 Objectives

- Analyze overall sales performance  
- Understand customer purchasing behavior  
- Identify top-performing products and categories  
- Evaluate delivery performance and delays  
- Analyze payment methods and trends  
- Measure customer satisfaction using reviews  

---

## 🗂️ Dataset Description

The dataset consists of multiple tables:

- Customers – Customer details  
- Orders – Order information and timestamps  
- Order Items – Product-level order details  
- Products – Product information  
- Sellers – Seller details  
- Payments – Payment type and value  
- Reviews – Customer feedback  
- Geolocation – Location mapping  
- Category Translation – Product category mapping  

---

## 🧹 Data Cleaning (Excel)

- Handled missing values  
- Removed duplicate records  
- Standardized date formats  
- Ensured consistent data structure  

---

## 🗄️ Data Modeling (SQL)

Created a structured data model using:

### ⭐ Dimension Tables
- dim_customers  
- dim_products  
- dim_sellers  

### ⭐ Fact Table
- fact_orders  

Used LEFT JOINS to combine multiple tables and avoid data loss.

---

## 🔄 Data Transformation

- Standardized state and city names  
- Translated product categories into English  
- Cleaned and grouped payment types  
- Ensured consistent data formatting  

---

## 📊 Key Metrics (KPIs)

- Total Revenue  
- Total Orders  
- Total Customers  
- Average Order Value (AOV)  
- Late Delivery Percentage  
- Average Review Score  

---

## 📈 Power BI Dashboard Features

The interactive dashboard includes:

- 📊 Monthly Sales Trend  
- 🌍 Revenue by State  
- 🏙️ Revenue by City  
- 🛍️ Revenue by Category  
- 💳 Payment Method Analysis  
- ⭐ Review Distribution  
- 🚚 Delivery Performance Analysis  
- 🏪 Seller Performance  

---

## 🔍 Key Insights

- Credit Card is the most used payment method  
- Certain categories generate the highest revenue  
- Delivery delays impact customer reviews  
- Major cities contribute most of the revenue  
- Seasonal trends affect sales performance
